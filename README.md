# Praktikum Struktur Data dan Algoritma

Repository ini berisi materi, tugas, dan panduan praktikum untuk mata kuliah **Struktur Data dan Algoritma**.

---

## Kontrak Praktikum

1. Praktikan wajib mengikuti seluruh kegiatan praktikum sesuai jadwal yang telah ditentukan.
2. Praktikan diwajibkan membawa laptop dan memastikan software yang diperlukan telah terinstall.
3. Kehadiran minimal **75%** dari total pertemuan praktikum.
4. Tugas praktikum wajib dikumpulkan sesuai dengan deadline yang diberikan.
5. Dilarang melakukan **plagiarisme** dalam bentuk apa pun.
6. Jika terdapat kendala selama praktikum, praktikan diharapkan segera menghubungi asisten praktikum.
7. Praktikan wajib menjaga ketertiban dan kondusivitas selama praktikum berlangsung.

---

## Penilaian Praktikum 

35% Nilai Tugas
25% Nilai Responsi 1
30% Nilai Responsi 2
10% Nilai Kehadiran
Keaktifan sebagai penambah nilai tugas dan nilai akhir
