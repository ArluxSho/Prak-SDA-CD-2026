## Tugas Praktikum Pertemuan 10

### Keterangan Tugas

Berdasarkan kode implementasi Dijkstra Java pada [materi](https://github.com/ArluxSho/Prak-SDA-CD-2026/blob/main/pertemuan-10/Shortest-Path.md), lengkapi kode sesuai yang telah dijelaskan pada praktikum jika diketahui bahwa:

### A. Kasus 1

Diketahui sebuah graph memiliki 6 vertex dan hubungan antar node:

- Node 0 ke 1 jaraknya 4
- Node 0 ke 2 jaraknya 2
- Node 1 ke 2 jaraknya 5
- Node 1 ke 3 jaraknya 10
- Node 2 ke 4 jaraknya 3
- Node 4 ke 3 jaraknya 4
- Node 3 ke 5 jaraknya 11
- Node 4 ke 5 jaraknya 2
- Node 2 ke 5 jaraknya 9
- Node 0 ke 5 jaraknya 20

### B. Kasus 2

Diketahui sebuah graph memiliki 7 vertex dan hubungan antar node:

- Node 0 ke 1 jaraknya 6
- Node 0 ke 2 jaraknya 1
- Node 1 ke 3 jaraknya 5
- Node 2 ke 1 jaraknya 2
- Node 2 ke 4 jaraknya 7
- Node 3 ke 5 jaraknya 3
- Node 4 ke 5 jaraknya 1
- Node 5 ke 6 jaraknya 4
- Node 1 ke 6 jaraknya 15
- Node 0 ke 6 jaraknya 25

Dengan data di atas, implementasikan kedalam kode lalu berikan analisis singkat dengan memberikan komentar di kodenya.
