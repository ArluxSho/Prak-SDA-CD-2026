Buatlah sebuah program menggunakan bahasa Java dengan tema bebas dengan memanfaatkan materi Struktur Data dan Algoritma.

### Ketentuan Implementasi Materi (50 Poin)

Pilih dan implementasikan **satu materi dari setiap kategori berikut**:

**Kategori 1 (10 poin)**

- Graph
- Disjoint Set

**Kategori 2 (20 poin)**

- Quick Sort
- Merge Sort

**Kategori 3 (20 poin)**

- Binary Search
- Minimum Spanning Tree
- Shortest Path

Contoh:

Jika mahasiswa memilih **Disjoint Set**, **Quick Sort**, dan **Shortest Path**, maka ketiga materi tersebut harus diimplementasikan dan digunakan secara nyata dalam program.

### Penilaian Tambahan (50 Poin)

**Fungsionalitas Program (0–20 poin)**

- Program berjalan dengan baik sesuai tujuan yang dibuat.
- Fitur yang disediakan dapat digunakan tanpa kesalahan yang signifikan.
- Implementasi materi algoritma terintegrasi dengan fungsi program.

**Ketepatan Struktur Data dan Analisis Kompleksitas (0–30 poin)**

- Pemilihan struktur data sesuai dengan kebutuhan program.
- Implementasi algoritma dilakukan dengan benar.
- Analisis kompleksitas waktu (Time Complexity) dan/atau ruang (Space Complexity) dijelaskan dengan tepat pada laporan.
