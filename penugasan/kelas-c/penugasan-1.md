# Tugas Praktikum SDA 1

## Penugasan

Berdasarkan materi **Java Introduction** yang telah dipelajari, buatlah sebuah **program Java sederhana dengan tema bebas** yang memanfaatkan konsep dasar pemrograman berikut:

### 1. Variabel dan Tipe Data
Gunakan beberapa tipe data dalam program seperti:
- `int`
- `double`
- `char`
- `String`

### 2. Operator
Gunakan operator dalam program, seperti:
- Operator aritmatika (`+`, `-`, `*`, `/`)
- Operator perbandingan (`>`, `<`, `==`)
- Operator logika (`&&`, `||`)

### 3. Java Control Flow
Gunakan struktur kontrol untuk mengatur jalannya program, seperti:
- `if - else`
- `switch`
- perulangan (`for`, `while`, atau `do-while`)

### 4. Array
Gunakan **array** untuk menyimpan beberapa data dalam program.

Program dapat berupa apa saja, misalnya:
- Program menghitung nilai
- Program kasir sederhana
- Program menghitung luas bangun
- Program data mahasiswa
- Program menu makanan
- atau program sederhana lainnya.

---

## Ketentuan

1. Program harus menyertakan **comment (`//`)** yang menjelaskan bagian-bagian kode.
2. Program harus dapat **dijalankan tanpa error**.
3. Program harus menggunakan **variabel, operator, control flow, dan array**.
4. Sertakan **screenshot hasil program ketika dijalankan**.
