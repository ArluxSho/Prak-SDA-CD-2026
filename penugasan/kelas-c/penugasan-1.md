# Tugas Praktikum SDA 1

## Penugasan

Dari materi java introduction yang telah dipelajari hari ini, buatlah sebuah program yang memuat 4 aspek berikut: 

1. Variabel dan tipe data

2. Operator

3. Java control flow

4. Array

## Ketentuan

Ketentuan dari tugas ini antara lain: 

1. Program harus menyertakan comment (// atau / /) yang menjelaskan bagian-bagian kode.

2. Program harus dapat dijalankan tanpa error.

3. Sertakan screenshot hasil program ketika dijalankan.

## Format Pengumpulan

Format file yang dikumpulkan adalah sebagai berikut:

1. File program Java dengan format nama: 

```
D_PSDA01_NIM_NAMA.java
```

Contoh: 
```
D_PSDA01_L0124085_AisyahNurulSholikhah.java
```

2. Screenshot hasil program berjalan.

## Deadline

Batas waktu untuk pengerjaan tugas ini adalah sampai dengan hari Senin, 16 Maret 2026. Keterlambatan akan mengurangi nilai sesuai kesepakatan.
