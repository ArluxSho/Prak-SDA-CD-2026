# Penugasan Praktikum – Pertemuan 2

## Materi
- List
- Stack
- Queue

## Deskripsi Tugas

Buatlah sebuah **program Java sederhana** yang menggunakan tiga struktur data berikut:

1. **List**  
   Digunakan untuk menyimpan beberapa data (misalnya nama, angka, atau barang).

2. **Stack**  
   Gunakan Stack untuk **menyimpan dan mengambil data terakhir**.

3. **Queue**  
   Gunakan Queue untuk **menyimpan data dalam bentuk antrian**.

Program minimal harus dapat:

- Menambahkan beberapa data ke **List**
- Menampilkan isi **List**
- Menambahkan data ke **Stack** lalu mengambil data terakhir
- Menambahkan data ke **Queue** lalu memproses antrian pertama

## Ketentuan

1. Program harus menggunakan:
   - `List` / `ArrayList`
   - `Stack`
   - `Queue`
2. Program harus dapat **dijalankan tanpa error**
3. Sertakan **comment (`//`) pada bagian penting kode**
4. Sertakan **screenshot hasil program ketika dijalankan**
