# PROGRAM PRIBADI 

## Deskripsi Umum

Untuk memperdalam pemahaman mahasiswa tentang materi Set dan Map, Mahasiswa diminta untuk membuat sistem sederhana untuk menyelesaikan suatu masalah yang memanfaatkan Set dan Map. Program harus bisa menyelesaikan suatu permasalahan entah mencari sesuatu atau menambah sesuatu. 

## Ketentuan Tugas ##

1. Mahasiswa bebas menentukan tema
2. menggunakan Set
3. menggunakan Map
4. Minimal menggunakan 4 operasi
   
## Format Pengumpulan

1. File program Java disatukan dalam folder/zip dengan format nama:

  `kelas_PSDA02_NIM_NAMA.zip`

Contoh:

  `C_PSDA02_L0124063_MuhammadIrfan.zip`

2. Screenshot hasil program dijalankan serta kesimpulan singkat analisis *runtime* dalam format PDF:

  `kelas_PSDA02_NIM_NAMA.pdf`

Contoh:

  `C_PSDA02_L0124063_MuhammadIrfan.pdf`
