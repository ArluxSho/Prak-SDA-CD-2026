# Latihan Pertemuan 10 dan 11

Materi:

* Pertemuan 10: Dijkstra dan Shortest Path
* Pertemuan 11: Bubble Sort, Selection Sort, dan Insertion Sort

---

## Soal 1 — Dijkstra

Diberikan graph berikut:

| Edge  | Bobot |
| ----- | ----: |
| A - B |     4 |
| A - C |     2 |
| B - D |     5 |
| C - D |     1 |
| D - E |     3 |

Gunakan algoritma **Dijkstra** untuk mencari jalur terpendek dari **A ke E**.

---

## Soal 2 — Dijkstra dengan PriorityQueue

Diberikan graph berikut:

| Dari | Ke | Bobot |
|---:|---:|---:|
| 0 | 1 | 4 |
| 0 | 2 | 2 |
| 1 | 3 | 5 |
| 2 | 3 | 1 |

Gunakan konsep **Dijkstra dengan PriorityQueue** untuk mencari jarak terpendek dari node `0` ke node `3`.

Tuliskan:
1. Urutan node yang diproses.
2. Perubahan jarak sementara.
3. Hasil jarak terpendek dari `0` ke `3`.

---

## Soal 3 — Bubble Sort

Diberikan array berikut:

```java
int[] data = {5, 3, 8, 1, 2};
```

Urutkan array tersebut secara ascending menggunakan **Bubble Sort**.

Tampilkan hasil array setelah setiap iterasi.

---

## Soal 4 — Selection Sort atau Insertion Sort

Diberikan array berikut:

```java
int[] nilai = {70, 60, 80, 65, 75};
```

Pilih salah satu algoritma berikut:

* Selection Sort
* Insertion Sort

Urutkan array tersebut secara ascending, lalu tuliskan langkah-langkah singkat proses pengurutannya.

---

## Catatan

Kerjakan setiap soal dengan menuliskan:

1. Algoritma yang digunakan.
2. Langkah penyelesaian.
3. Hasil akhir.
