# Quick Sort

## Pengertian

Quick Sort adalah salah satu algoritma pengurutan data (sorting) yang paling efisien dengan kompleksitas O(N log N). Inti dari Quick Sort adalah penggunaan strategi Divide and Conquer dengan memanfaatkan pivot. Meski sangat cepat secara rata-rata, Quick Sort memiliki tantangan tersendiri: kinerjanya sangat bergantung pada pemilihan "titik tumpu" atau pivot. Jika titik tumpu ini selalu terpilih secara buruk pada setiap putaran, kecepatannya bisa menurun drastis menjadi $O(N^2)$.

## Cara Kerja

Cara kerja Quick Sort berpusat pada satu elemen krusial yang disebut Pivot (Titik Tumpu). Berikut adalah langkah-langkah utamanya:

1. Pilih Pivot
   Algoritma akan memilih satu elemen dari array untuk dijadikan pivot. Pivot ini bisa diambil dari elemen pertama, terakhir, tengah, atau bahkan acak. Tapi disini kita akan ambil terakhir
2. Melakukan Partisi
   Algoritma akan menyusun ulang array dengan aturan berikut:
   - Semua angka yang lebih kecil dari pivot dipindahkan ke sebelah kiri pivot.
   - Semua angka yang lebih besar dari pivot dipindahkan ke sebelah kanan pivot.
     Setelah proses partisi selesai, pivot tersebut dijamin sudah berada di posisi akhir yang benar (sudah terurut), meskipun elemen di kiri dan kanannya mungkin belum terurut sempurna.
3. Lakukan Divide & Conquer (Rekursi)
   Sekarang, kita memiliki dua sub-array: sub-array kiri (lebih kecil dari pivot) dan sub-array kanan (lebih besar dari pivot).
   Algoritma akan memanggil dirinya sendiri (rekursi) untuk mengulangi Langkah 1 dan 2 pada sub-array kiri dan kanan tersebut. Proses ini berulang sampai ukuran sub-array hanya tersisa 1 elemen atau kosong.

## Contoh Proses Quick Sort:

```

Array Awal: 23  56  9  103  77

Iterasi 1 (Memproses Seluruh Array):
Pilih Pivot (elemen terakhir): 23  56  9  103  [77]
Pindahkan angka < 77 ke kiri, dan angka > 77 ke kanan.
Hasil Partisi: 23  56  9  [77]  103

| Kiri (< 77) | Pivot (Fix) | Kanan (> 77) |
| ----------- | ----------- | ------------ |
| 23  56  9   | 77          | 103          |

Iterasi 2 (Memproses sub-array KIRI dari Iterasi 1: 23, 56, 9):
Pilih Pivot (elemen terakhir): 23  56  [9]
Pindahkan angka < 9 ke kiri, dan angka > 9 ke kanan.
Hasil Partisi: [9]  56  23

| Kiri (< 9) | Pivot (Fix) | Kanan (> 9) |
| ---------- | ----------- | ----------- |
| (kosong)   | 9           | 56  23      |

Iterasi 3 (Memproses sub-array KANAN dari Iterasi 2: 56, 23):
Pilih Pivot (elemen terakhir): 56  [23]
Pindahkan angka < 23 ke kiri, dan angka > 23 ke kanan.
Hasil Partisi: [23]  56

| Kiri (< 23) | Pivot (Fix) | Kanan (> 23) |
| ----------- | ----------- | ------------ |
| (kosong)    | 23          | 56           |


*Catatan: Sub-array yang hanya berisi 1 elemen (seperti 103 pada Iterasi 1, dan 56 pada Iterasi 3) secara otomatis dianggap sudah terurut (Fix) dan tidak perlu dipartisi lagi.

Hasil akhir -> 9, 23, 56, 77, 103
```

## Implementasi dalam Java

```java
public class QuickSort {

    // Metode utama untuk menjalankan rekursi
    public void sort(int[] arr, int low, int high) {
        if (low < high) {
            // Dapatkan indeks partisi (posisi pivot yang sudah benar)
            int pi = partition(arr, low, high);

            // Urutkan elemen secara rekursif di sebelah KIRI pivot
            sort(arr, low, pi - 1);

            // Urutkan elemen secara rekursif di sebelah KANAN pivot
            sort(arr, pi + 1, high);
        }
    }

    // Metode untuk melakukan partisi array
    private int partition(int[] arr, int low, int high) {
        // Kita jadikan elemen paling akhir sebagai pivot
        int pivot = arr[high];

        // Indeks untuk elemen yang lebih kecil dari pivot
        int i = (low - 1);

        for (int j = low; j < high; j++) {
            // Jika elemen saat ini lebih kecil atau sama dengan pivot
            if (arr[j] <= pivot) {
                i++; // Majukan indeks elemen yang lebih kecil

                // Tukar arr[i] dengan arr[j]
                int temp = arr[i];
                arr[i] = arr[j];
                arr[j] = temp;
            }
        }

        // Terakhir, tukar elemen pivot (arr[high]) dengan elemen di posisi (i + 1)
        // Agar pivot berada di tengah-tengah kelompok kecil dan besar
        int temp = arr[i + 1];
        arr[i + 1] = arr[high];
        arr[high] = temp;

        // Kembalikan posisi indeks pivot yang baru
        return i + 1;
    }

    // Metode bantuan untuk mencetak array ke console
    public static void printArray(int[] arr) {
        for (int j : arr) {
            System.out.print(j + " ");
        }
        System.out.println();
    }

    // Main method untuk menjalankan program
    public static void main(String[] args) {
        int[] data = {10, 80, 30, 90, 40, 50, 70};

        System.out.println("Array sebelum diurutkan:");
        printArray(data);

        QuickSort qs = new QuickSort();
        // Mengirim array, indeks awal (0), dan indeks akhir (panjang - 1)
        qs.sort(data, 0, data.length - 1);

        System.out.println("\nArray setelah diurutkan (Quick Sort):");
        printArray(data);
    }

}
```
